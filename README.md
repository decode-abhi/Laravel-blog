# Blog Project

This is a Laravel-based blog project demonstrating CRUD, authentication, and more.

## Features
- User authentication
- Create, read, update, delete blog posts
- Pagination
- Image upload

## Setup
- `git clone`
- `composer install`
- `cp .env.example .env`
- `php artisan key:generate`
- `php artisan migrate`
- `php artisan serve`
